# js正则替换日期格式

## 2018-1-1  ===> 2018-01-01
```
'2017-6-5'.replace(/-(\d{1})/g,'-0$1') // 匹配 - 后是一位的数字，替换成 -0 形式
// "2017-06-05"

这个替换规则应该是把-(\d{1})整个替换了，分组是为后面$1用的
```
